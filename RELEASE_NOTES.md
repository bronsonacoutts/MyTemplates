# Release Notes — v1.0.0

**Release Date:** 2024-01-01
**Tag:** `v1.0.0`

---

## Changes

- **Initial release** of the MyTemplates TypeScript/Node.js project template.
- Complete project scaffolding with ESLint, Prettier, TypeScript strict mode, and Vitest.
- Husky git hooks enforce code quality at commit and push time.
- GitHub Actions CI pipeline with lint, type-check, test coverage, and build jobs.
- CodeQL security scanning configured for weekly and on-push analysis.
- Automated PR validation for branch names and commit message conventions.
- Release workflow triggered by version tags (`v*.*.*`).
- Comprehensive developer documentation templates in `docs/`.
- AI coding guidelines in `agent-instructions.md` synced to `.github/copilot-instructions.md`.
- Security policy, Code of Conduct, and MIT License added.
